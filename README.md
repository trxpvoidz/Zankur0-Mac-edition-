# INSTALLATION 
**METHOD 1 (for older mac os versions)**

Locate the .app file in Finder.
Right-click (or Control-click) the app and choose Open.
The dialog shows an Open button. Click it.

**METHOD 2**

System Settings → Privacy & Security and scroll down to find an "Open Anyway" button after your first failed launch attempt.

**METHOD 3**

to open Zankur0 with terminal you use this command line
```sudo xattr -dr com.apple.quarantine /path/to/Zankur0.app```
